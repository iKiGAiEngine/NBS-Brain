# Lessons Learned Index

A lesson is useful only when it changes a future decision, process, rule, checklist, training item, or system behavior.

Capture:

- what happened and the evidence
- expected versus actual outcome
- root and contributing causes
- financial, schedule, customer, and workload impact
- corrective action and owner
- process, rule, or AiPM change required
- validation that the improvement worked
